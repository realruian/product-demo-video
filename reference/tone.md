# Tone — 调性规范

这是产品演示片「高级、克制、真实」质感的来源。**优先遵守目标项目自己的 DESIGN.md**；项目没规定时用以下默认。

## 配色（默认极简，不引彩色）
| 用途 | 值 |
|---|---|
| 画布背景 | `#F4F4F6` + 顶部柔和径向高光 |
| 窗口/卡片 | `#FFFFFF` |
| 边框 | `rgba(20,20,20,0.06~0.08)` |
| chip / 浅底 | `#F0F0F2` / `#F2F2F2` |
| 主文字 | `#141414` / `#191919` |
| 次文字 | `#6B6B6B` / `#8A8A90` |
| 弱文字 | `#9A9AA0` |
| 功能红（录制点/连接断/结束键）| `#E5484D` —— 只在真实 UI 里有时保留 |
| 功能绿（通过/正分）| `#2E9E5B` —— 同上 |

**不要**为了好看引入品牌色/渐变色；浏览器红绿灯点也改成灰 `#DCDCE1`。

## 字体
- 中英文统一 **PingFang**：`"PingFang SC","Heiti SC",sans-serif`。渲染机为 macOS 直接可用。
- 标题 `fontWeight 600`（PingFang 最粗 Semibold），正文 `400`。
- **不引 Inter 等拉丁字体**配中文（fallback 不一致，且无谓依赖）。

## 浏览器窗口 mockup
- 圆角 18，白底，柔和双层投影 `0 24px 60px rgba(20,20,20,.12), 0 4px 14px rgba(20,20,20,.06)`。
- 顶部 chrome bar 高 46：三个灰圆点 + 地址栏 pill（`{addr}` 随场景变）。
- 内容区比例 16:10（按 1440×900 截图等比），截图 `objectFit:"contain"` 完整显示。

## 运镜（缓动统一）
- 缓动：`Easing.bezier(0.16,1,0.3,1)`（高级 ease-out）。
- 窗口浮入：`spring({damping:200, mass:0.8})` + translateY 64→0。
- Ken Burns：整窗 `scale 1→1.03`（**作用在窗口外层，不是内部图片**，否则裁页面内容）。
- 场景 crossfade：opacity 在场景边界 ±6 帧交叉。
- 收尾：窗口淡出缩小 0.94 + logo 淡入。

## 节奏
- 默认每段统一 **60 帧 / 2s @30fps**，整齐。
- 内容多的代码界面（面试/报告）可放宽到 72~75 帧。
- 进阶「卡点」：测 BGM 的 BPM，让每段 = 整数个小节（`帧数 = round(60/BPM * 拍数 * fps)`）。

## 字幕
- 底部居中，标题 PingFang `fontSize 62 / weight 600 / letterSpacing -1`，副标题 `fontSize 31 / weight 400 / 灰`。
- 跟随场景 crossfade + 轻微上移入场。

## Logo
- 用项目**真实矢量字标**（找 `public/icons/logo.svg` 之类），顶部小 + 收尾大。
- 绝不用纯文字假冒 logo。

## 画布尺寸预设
| 比例 | 尺寸 | 场景 |
|---|---|---|
| 竖版 3:4（默认）| 1280×1706 | 抖音/小红书/Reels |
| 方版 1:1 | 1080×1080 | 朋友圈/IG feed |
| 横版 16:9 | 1920×1080 | YouTube/B站/官网/PPT |

横版时窗口可更宽、字幕字号相应缩小。

## 声音
- **BGM**：Mixkit/Pixabay 免版权可商用免署名。整片铺底，音量 ~0.85，首尾淡入淡出。有节奏感选 `uplifting`/`energetic` mood，但避免 Rock/Metal 太吵盖过极简调性。
- **TTS**：AiHubMix/OpenAI `gpt-4o-mini-tts`（真人级，可 prompt 控语气）。**不用 macOS `say`**（机械）。每句对齐场景，场景时长按旁白长度。
- 旁白 + 字幕同时存在时，文案可一致或互补。

## 反面清单（出现即降级为「AI 味」）
- 彩色乱用 / 渐变滥用 / 红绿灯彩点
- Inter 等拉丁字体配中文
- `objectFit:cover` + 放大 → 裁掉页面内容
- 凭想象画截不到的页面（必须读真实组件代码）
- macOS say 机械音旁白
- 节奏忽快忽慢、字幕字号飘忽
- 纯文字假 logo
- emoji
